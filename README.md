# Asteroid Dodger

Simple, lightweight 🕸 game.

[![Play Game](https://img.shields.io/badge/Play-Now-blue?style=for-the-badge)](https://amiralimahmoodi.github.io/asteroid-dodger/)

## How to Play

- **Left:** `←` or `A`
- **Right:** `→` or `D`
- **Start/Restart:** `Space`
